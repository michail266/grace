Αυτή είναι η εργασία των μεταγλωττιστών του 2025, στην οποία μας ζητήθηκε να υλοποιήσουμε έναν μεταγλωττιστή για τη γλώσσα Grace.

Πρόκειται για το πρώτο παραδοτέο, στο οποίο υλοποιήσαμε τη λεκτική και συντακτική ανάλυση.

Το πρόγραμμα εκτελείται με τις εντολές που περιλαμβάνονται στο αρχείο test.txt. Μπορείτε να προσθέσετε εντολές, και ελπίζουμε να τις εκτελεί σωστά.


Στον lexer έχουμε τοποθετήσει τις περισσότερες λεκτικές μονάδες (tokens), ενώ στον parser υλοποιήσαμε κάποιες βασικές λειτουργίες όπως:

    Βασικές πράξεις

    Ανάθεση μεταβλητής

    Εκτύπωση με print (όχι puts, puti)

    Αναγνώριση επιπλέον δομών από τον parser, όπως: if then else, while do, return, prints.


Λειτουργίες που δεν υλοποιήσαμε: 

    fun (function) types (var int string) και το prints(print string)
